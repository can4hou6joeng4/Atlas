# README Assets

Place README media assets here, including PNG screenshots, app icons, and GIF demos. Use lowercase, dash-separated filenames such as `network-debugger.png` or `notch-island-demo.gif`.

Screenshot files and GIF demos used by the README live in `screens/`.

Current README hero and scenario assets:

- `screens/readme-main-overview.png` - primary TokenAtlas dashboard visual.
- `screens/readme-usage.png` - Usage scenario visual.
- `screens/readme-sessions.png` - Sessions scenario visual.
- `screens/readme-git-activity.png` - Git Activity scenario visual.

These files are captured from the running macOS app via the canonical debug
build flow. Do not replace them with mockups; crop or anonymize real captures
when needed so private local project data is not exposed in the public README.
